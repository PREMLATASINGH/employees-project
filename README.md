# Employees Project

Small Python project for working with an employees CSV file.

## Contents
- `employees.py` — main script that processes the CSV.
- `employees_1000.csv` — sample employee dataset.

## Requirements
- Python 3.8+
- (No external packages required unless `employees.py` uses them.)

## Run
Open a terminal in the project folder and run:

```bash
python employees.py
```

If `employees.py` expects the CSV as an argument, run:

```bash
python employees.py employees_1000.csv
```
